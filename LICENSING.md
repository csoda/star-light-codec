# Licensing Policy

Star Light Codec follows the same licensing policy as Star Light.

| Material | Default license |
| --- | --- |
| Code, scripts, tests | Apache-2.0 |
| Documentation, specifications, roadmap text | CC BY 4.0 |
| Small fixtures, sample metadata, reusable examples | CC0-1.0 |

The root [LICENSE](LICENSE) file contains the Apache-2.0 license for code.
The [LICENSES](LICENSES/) directory contains short SPDX-style pointers for the
documentation and sample-data licenses used by this repository.

This repository is intended for broad public reuse. If a file needs a different
license, add a short note in that file or in this document.
